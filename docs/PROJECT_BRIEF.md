# Краткое описание проекта

## Название

T-Invest Signal Bot

## Цель

Production-grade signal scanner на базе T-Invest API.  
Сканирует watchlist, анализирует рынок, отправляет качественные BUY/SELL/ROCKET сигналы в Telegram.

## Фаза 1 (текущая)

- Только `read-only` токен
- Никаких реальных заявок
- Никакой автоторговли
- Scanner + Analyzer + Telegram notifier

## Будущие фазы

- Фаза 2: Paper trading / sandbox
- Фаза 3: Автоторговля (отдельное решение)

## Запреты фазы 1

- `TINVEST_FULL_ACCESS_TOKEN` — запрещён
- Реальные orders — запрещены
- `TradingExecutor` — только интерфейс/stub, disabled

## Формат сигнала

```
🟢 $TICKER BUY

Вход: X–Y ₽
Стоп: Z ₽
Цели: A / B / C ₽
R/R: 1:N
Доля: до N%
Срок: краткосрок / 1–3 дня
Риск: низкий / средний / высокий

Причина: причина 1 + причина 2
```

Максимум 8–10 строк. R/R ниже 2 — не отправлять.

## Правила сигналов

- Нет entry/stop/take → нет сигнала
- R/R < 2 → нет сигнала
- Дубль по ticker/setup → нет сигнала
- Нет объёмного подтверждения → нет сигнала
- Все no-signal причины → в логи/БД

## Предположения

1. T-Invest API предоставляет свечи, котировки и статус торгов через официальный Python SDK (`tinkoff-invest-python-grpc-client` или `tinkoff.invest`).
2. Аномальный объём определяется как объём > N×(средний объём за M периодов), N и M конфигурируются.
3. ATR считается по свечам выбранного таймфрейма (по умолчанию 1 день).
4. MockNewsProvider всегда возвращает нейтральные новости.
5. NVIDIA AI API использует OpenAI-совместимый интерфейс (`openai` SDK с custom base_url).
6. Сигнал считается дублем, если по тому же ticker + direction уже был сигнал в последние N часов (конфигурируется).
